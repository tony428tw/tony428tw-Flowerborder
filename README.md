# 花境設計小幫手 Flower Border Design Helper

一個可直接在瀏覽器開啟的花境植栽設計輔助工具，協助景觀設計、校園綠化、庭園花境與公共空間植栽配置快速產出初步方案。

Designed by Jerry Hsu

## 功能特色

- 依基地條件產生花境設計策略
- 自動建立植栽表與逐株分析
- 支援低維護、誘蝶蜜源、鳥類食源與台灣綠化規範考量
- 產生平面圖、剖立面圖、軸側圖與分析海報的圖片提示詞
- 支援一鍵複製提示詞
- 支援植栽表匯出 CSV
- 可下載完整 Markdown 方案
- 純前端靜態網頁，不需要 API key、後端伺服器或資料庫

## 使用方式

1. 填寫基地位置、基地尺寸、設計場域、設計風格、日照、土壤排水與色彩方向。
2. 選擇是否需要低維護、誘蝶蜜源、鳥類食源與台灣綠化規範考量。
3. 勾選希望輸出的內容。
4. 點擊「產生花境方案」。
5. 在不同頁籤查看：
   - 總覽
   - 植栽表
   - 圖片提示詞
   - 檢核
6. 可複製提示詞、匯出 CSV 或下載 Markdown。

## 植栽資料來源

本專案使用臺北市政府全球資訊網 `附表3-綠化植栽種類.pdf` 作為植栽資料庫基礎，並整理出 `data/plant_database_seed.csv` 作為前期參考資料。

植物選用仍應依實際基地條件、氣候、土壤、排水、日照、安全性與維護能力進行專業判斷。
