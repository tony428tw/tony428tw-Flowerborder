# 姓名戰鬥陀螺仿製版

這是一個純前端仿製作品，可直接用瀏覽器開啟 `index.html`。

## 原作畫面分析

- 介面主軸是「紙張格線 + 毛筆字 + 紅黑雙方」。
- 玩法核心是把名字拆成單字，每個字都變成一顆陀螺。
- 三字姓名模式會進行完整三回合，第一字對第一字、第二字對第二字、第三字對第三字。
- 單字模式則只打一回合。
- 戰鬥畫面由左右選手欄、中間圓形陀螺場、回合進度、完整度與戰鬥紀錄組成。

## 仿製規則

- 每個字會依照 Unicode code point 產生固定能力值：重量、速度、攻擊、防禦、穩定。
- 同一個字在同一位置會得到固定數值，因此同一組名字重玩仍有一致的角色感。
- 每回合用能力值加上少量字種差異計算勝者，回合結束後會暫停。
- 玩家必須按下「下一回合」才會進到下一個字的對戰。
- Canvas 會模擬兩顆 2.5D 字形陀螺在圓形場地內旋轉、碰撞、減速，並留下筆畫碎片。
- 下方紀錄會顯示本回合對戰字、撞擊事件、中心旋轉力耗盡、勝者與完整度。
- 勝負與歷史上場紀錄會存入 `localStorage`。

## 檔案

- `index.html`：頁面結構。
- `styles.css`：紙張格線、字卡、競技場與響應式排版。
- `script.js`：姓名拆字、能力生成、回合判定、Canvas 動畫與紀錄。
