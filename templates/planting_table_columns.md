# 植栽表欄位

| 欄位 | 說明 |
| --- | --- |
| 編號 | P-01、T-01、S-01、G-01 等 |
| 植物中文名 | 如樟樹、杜鵑、玉龍草 |
| 學名 | 避免名稱爭議 |
| 類型 | 喬木、灌木、藤蔓、草花、地被、草皮、水生／濕生植物 |
| 原生／外來 | 原生、歸化、外來或待確認 |
| 高度 | 成熟高度與設計高度 |
| 冠幅／株距 | 影響配置密度 |
| 花色 | 花境視覺重點 |
| 花期 | 春、夏、秋、冬 |
| 葉色／質感 | 細葉、闊葉、斑葉、線形葉 |
| 日照需求 | 全日照、半日照、耐陰 |
| 水分需求 | 耐旱、中等、濕生 |
| 維護難度 | 低、中、高 |
| 生態功能 | 蜜源、誘蝶、鳥類食源 |
| 景觀功能 | 背景、骨架、焦點、邊界、地被 |
| 設計位置 | 前景、中景、背景、節點 |
| 注意事項 | 有毒、落果、竄根、斷枝、刺、過敏等 |
