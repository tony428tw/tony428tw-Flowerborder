# 品質檢核表

| 檢核項目 | 檢查重點 |
| --- | --- |
| 適地適種 | 是否符合台灣氣候、日照與水分 |
| 高度合理 | 是否前低後高、層次清楚 |
| 季節連續 | 是否四季皆有觀賞重點 |
| 維護可行 | 是否避免過度高維護植物 |
| 安全性 | 是否避開有毒、落果、竄根、尖刺植物 |
| 生態性 | 是否提供蜜源、誘蝶、鳥類食源 |
| 圖面一致 | 植栽表、照片、平面、剖面、軸側是否相互對應 |
| 專業性 | 是否有比例、北箭、圖例、植物編號與分析 |

## 高風險提醒

- 落果：第倫桃等不宜靠近主要人行動線或停車區。
- 竄根：榕樹等需避開建築基礎、鋪面、管線與狹窄花台。
- 斷枝：黑板樹等需評估風害與人流安全。
- 有毒：海檬果等不宜用於幼兒、校園或高接觸公共空間。
